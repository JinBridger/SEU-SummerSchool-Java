## ✨ 快速开始

### Requirements

- `JDK >= 17`
- `MySQL >= 8.0.0`

### 启动方法

将 Release 中的服务器 SQL 部署至服务器/本机，同时在本机运行 server 与 client 的 JAR 文件即可
